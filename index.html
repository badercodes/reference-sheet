Bader's Coding Reference Sheet

HTML Tables

Syntax Name Description
<table> Table Wrapper declaration for an HTML table
<thead> Table Head Wrapper declaration for a table's header, nested within it is the row with all the table headings
<th> Table Head Cell Equivalent to <td> for regular cells, this tag is used within <thead> to declare each cell of the header.
<tbody> Table Body Wrapper declaration for a table's body or main content, nested within it are the rows with all the table data
<tr> Table Row Defines a row in the table, nested within it will be the cells which form the columns
<td> Table Data Defines a cell within the table - this is nested within a row, the content nested within this element will form the contents of the column.
<tfooter> Table Footer Defines a the set of content that will form the footer of the table. This is primarily semantic HTML5 and not necessarily required for each table.

Table Tag Attributes

Syntax Name Description
colspan Column Span Specifies how many columns does the <td> span, in event we want a field to span more than 1 column
rowspan Row Span Specifies how many rows does the <td> span, in event we want a field to span more than 1 row
